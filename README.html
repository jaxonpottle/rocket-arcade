Run locally

Because the homepage uses fetch() for games/games.json, you need a local server (not file://).

Python

  • Windows: py -m http.server 8000
  • Mac/Linux: python3 -m http.server 8000
  • Open http://localhost:8000

Add a new game

  1. Copy games/_template/ ‚Üí games/<your-game-id>/
  2. Edit index.html and game.js
  3. Add an entry to games/games.json

Deploy (GitHub Pages)

  1. Create a GitHub repo and push this folder
  2. Repo Settings ‚Üí Pages ‚Üí Source: main + /root
  3. Your site is live

Run locally

Use a local server (not file://), because the homepage loads games/games.json.

  • Windows: py -m http.server 8000
  • Mac/Linux: python3 -m http.server 8000
  • Open http://localhost:8000

Deploy (GitHub Pages)

  1. Repo Settings ‚Üí Pages
  2. Source: Deploy from a branch
  3. Branch: main, Folder: / (root)

Add a new game

  1. Copy games/_template/ ‚Üí games/<your-game-id>/
  2. Build the game in game.js
  3. Add an entry to games/games.json
